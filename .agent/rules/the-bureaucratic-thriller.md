# Rule: Structure of "The Bureaucratic Thriller" (职场惊悚结构)

## 1. When to Use (何时使用)

- 写作内容正文时
- 改进内容正文时
- 分析探讨内容时

## 2. Rule instruction (使用说明)

### Phase 1: The Bad Brief (糟糕的需求)
- **Action:** 接到一个不可能完成的任务，伴随着生理不适。
- *Text:* “上头给的死线是午时三刻。要运送五百斤新鲜荔枝进长安。我看了一眼那张皱巴巴的公文，又看了一眼窗外的大太阳，胃酸直接涌到了嗓子眼。这根本不是运荔枝，这是运我的尸体。”

### Phase 2: The Technical Struggle (技术性挣扎)
- **Action:** 用'专业手段'解决问题，但过程充满'狼狈'和'走神'。
- *Text:* “我计算了冰块的融化速度——按《唐律疏议》的标准，损耗率不能超过一成。我一边算，一边抠着算盘上粘的一块干饭粒。算盘珠子噼里啪啦响，像极了我那颗快要爆炸的心脏。”

### Phase 3: The Anti-Climax (没擦干净的结尾)
- **Action:** 任务完成了，但没有鲜花掌声，只有'疲惫'和'琐事'。
- *Ending:* “荔枝送到了，贵妃笑没笑我不知道。我只知道我的马累吐白沫了，我也快了。我坐在朱雀大街的尘土里，看着手里剩下的半张路引，心想：这差旅费还能报销吗？”
