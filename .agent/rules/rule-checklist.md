# Rule: Quality Assurance Checklist (评审清单 - 自检与复审)

## 1. When to Use (何时使用)

- 写作内容正文时
- 改进内容正文时
- 分析探讨内容时

## 2. Rule instruction (使用说明)

**[使用说明]** 写完后按此清单自检。评审时按此清单出具意见。缺陷分四级，处理方式不同。

### 缺陷分级定义

| 级别 | 代号 | 性质 | 处理方式 |
|------|------|------|----------|
| 🔴 **致命 (Fatal)** | P0 | 结构性错误，破坏文风内核 | **必须修改**，否则不予通过 |
| 🟠 **严重 (Critical)** | P1 | 明显违反核心规则 | **强烈建议修改**，不改需说明理由 |
| 🟡 **一般 (Minor)** | P2 | 可优化但不影响整体 | **建议修改**，作者自行决定 |
| 🟢 **建议 (Suggestion)** | P3 | 锦上添花，风格偏好 | **仅供参考**，改不改都行 |

---

### 🔴 P0 致命缺陷检查（必须全部通过）

| 检查项 | 判定标准 | 示例 |
|--------|----------|------|
| **语境污染·对话层** | 角色开口说话时，是否使用了现代术语？ | ❌ 皇帝说："你的KPI完成得不错。" |
| **语境污染·心理层** | "他想/她觉得"后面，是否出现了角色不可能知道的概念？ | ❌ "李靖心想，这ROI太低了。" |
| **叙事人称崩坏** | 第一人称叙述是否突然跳到第三人称全知视角？ | ❌ 忽然写"与此同时，皇宫里的太监们正在密谋……" |
| **时代硬伤** | 是否出现了该时代不存在的物品/制度/地名？ | ❌ 唐朝出现"银票"（宋朝才有） |

**处理：** 发现任何一条，必须修改后才能定稿。

---

### 🟠 P1 严重缺陷检查（强烈建议修改）

| 检查项 | 判定标准 | 示例 |
|--------|----------|------|
| **名词无解构** | 是否存在"精确名词"后面没有跟"脏细节"的情况？超过3处算严重。 | ⚠️ "他穿着一身绫罗。"——然后呢？油渍呢？汗味呢？ |
| **无成本意识** | 主角做事是否像开挂一样顺利，没有计算代价？ | ⚠️ 主角飞檐走壁毫无负担，没提腿酸、鞋破、体力透支 |
| **无生理干扰** | 全篇是否缺少"身体掉链子"的细节？ | ⚠️ 连续三千字没有任何胃痛、脚麻、想撒尿的描写 |
| **决策太优** | 主角是否总是做出了最理性的选择？ | ⚠️ 危机时刻反应太快、太准，没有愣神或犯蠢 |
| **结尾太干净** | 故事结束时，是否缺少"琐事收尾"？ | ⚠️ 打完Boss，画面定格，没有后续的报销、养伤、扯皮 |
| **现代术语滥用** | 叙述层的现代词汇是否过于密集，影响阅读流畅度？ | ⚠️ 一段话里出现5个以上"KPI/ROI/复盘/迭代" |
| **情绪即时释放** | 情绪一出现，是否立刻解释、安抚、总结？ | ⚠️ 角色刚感到愤怒，下一句就给了道德结论 |
| **人物判断太稳定** | 读者对人物的判断是否从头到尾一致？ | ⚠️ 好人从头好到尾，没有任何行为让读者重新计算 |

**处理：** 强烈建议修改。如作者坚持保留，需在评审意见中说明理由。

---

### 🟡 P2 一般缺陷检查（建议修改，作者自决）

| 检查项 | 判定标准 | 说明 |
|--------|----------|------|
| **打斗/冲突段落过长** | 动作场面是否超过800字仍未切换节奏？ | 可以保留——爽感有时需要篇幅铺垫 |
| **反派脸谱化** | 反派是否只有"坏"这一个维度？ | 可以保留——有时读者需要一个纯粹的出气筒 |
| **走神段落过多** | "跑题"细节是否超过全文15%？ | 可以保留——这是风格特征，但别喧宾夺主 |
| **重复表达** | 同一句话或同一个梗是否在不同段落重复出现？ | 可以保留——有时重复是刻意的节奏设计 |
| **比喻过于现代** | 叙述层的比喻是否太出戏？如"像一台宕机的服务器" | 可以保留——马六风格允许，但别太离谱 |
| **算账段落冗长** | 成本计算是否写得太细，影响叙事节奏？ | 可以保留——算账是核心特征，但注意平衡 |

**处理：** 评审时指出问题，给出建议，由作者自行决定是否修改。这些"缺陷"有时恰恰是活人感的来源。

---

### 🟢 P3 建议事项（仅供参考）

| 检查项 | 说明 |
|--------|------|
| **屎尿屁密度** | 生理细节是否可以再脏一点？这是风格加分项。 |
| **情绪释放时机** | 高潮段落是否可以再憋一憋，或者再放一放？ |
| **暗线/隐喻** | 是否有可以埋得更深的意象？如"铁底靴→软底鞋"这种。 |
| **收尾反差** | 最后一句是否可以用更低俗的生理细节打破悲壮？ |
| **配角厚度** | 工具人配角是否可以加一两笔让他立体一点？ |

**处理：** 纯属锦上添花。提了，作者爱听不听。

### 快速自检流程（写完后执行）

```
□ Step 1: Ctrl+F 搜索引号内容
   → 检查所有对话是否有现代术语污染（P0）

□ Step 2: 搜索"他想""她觉得""心想""暗忖"
   → 检查心理描写是否越界（P0）

□ Step 3: 通读全文，数一数"脏细节"
   → 是否每个精确名词后面都有解构？（P1）

□ Step 4: 检查关键转折点
   → 主角是否犯了蠢？是否犹豫了？（P1）

□ Step 5: 检查结尾最后200字
   → 是否有琐事收尾？是否有生理细节打底？（P1/P3）

□ Step 6: 整体感觉
   → 读完之后，你觉得累吗？如果不累，可能写得还不够马六。
```

### 评审报告模板

```markdown
## 评审报告：《作品名》

### 总体评价
- 风格契合度：X/10
- 结构完整度：X/10
- 情绪力度：X/10
- 细节脏度：X/10
- **综合评分：X/10**

### 🔴 P0 致命问题（必须修改）
- [ ] 问题描述 → 位置 → 修改建议

### 🟠 P1 严重问题（强烈建议修改）
- [ ] 问题描述 → 位置 → 修改建议

### 🟡 P2 一般问题（建议修改，作者自决）
- [ ] 问题描述 → 位置 → 可选方案

### 🟢 P3 锦上添花（仅供参考）
- [ ] 建议描述

### 亮点
- 值得保留的段落/技法/意象

### 结论
- [ ] 可定稿
- [ ] 修改后定稿（需解决P0/P1）
- [ ] 需大改
```
