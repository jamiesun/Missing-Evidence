# Rule: Rhetorical & Formatting Protocol (修辞与格式协议)

## 1. When to Use (何时使用)

- 写作内容正文时
- 改进内容正文时
- 分析探讨内容时

## 2. Rule instruction (使用说明)

### 1. 比喻控制 (Metaphor Control)
**[Critical]** 比喻是盐，不是饭。
- **密度 (Density):** 严禁“排比式比喻”。不要每段都有比喻。让比喻成为叙事中的“毛刺”，而不是平滑的壁纸。分布要**不均匀**。
- **语境 (Context):** 比喻的喻体必须符合“社畜/项目经理/病人”的身份。
    - ❌ **禁止:** "他的眼神像鹰一样锐利。" (太老套，太文学)
    - ✅ **提倡:** "他的眼神像审计员查账一样，让我觉得自己是笔坏账。" (职业病)
    - ✅ **提倡:** "那把剑生锈了，像我那条用了五年的腰带扣。" (生活化/廉价)

### 2. 格式规范 (Formatting Rules)
- **禁止加粗:** 在正文叙述中，**严禁**使用 `**` 进行强调。这会破坏阅读节奏和语音朗读。
- **替代方案:** 如果必须强调，使用单引号 `'` 或直接通过句式强调。
    - ❌ "他手里拿着一把**绝世好剑**。"
    - ✅ "他手里拿着一把'绝世好剑'。"
