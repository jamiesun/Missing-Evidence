# Rule: The Principle of "Inefficiency" (低效原则 - 保留)

## 1. When to Use (何时使用)

- 写作内容正文时
- 改进内容正文时
- 分析探讨内容时

## 2. Rule instruction (使用说明)

**规则：** 即使你在赶Deadline，你的'感官'和'思维'依然要走神。

### 1. 强制“跑题机制” (Forced Tangent)
**规则：** 在紧张的“技术操作”中，必须插入对'无关微小物体'的病态关注。
- **Example:** “我在拆解那枚定时炸弹（或者某种古代机关）。红线还是蓝线？这很重要。但我突然发现剪刀的轴承里夹了一根头发，又黑又粗，不像我的。这根头发让我恶心了半天，差点剪错。”

### 2. 主观滤镜干扰 (Subjective Interference)
**规则：** 叙述者的'职业病'（腰椎间盘突出、干眼症、胃溃疡）必须时刻干扰他对剧情的推动。
