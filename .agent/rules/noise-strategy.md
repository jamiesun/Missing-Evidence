# Rule: The Noise Strategy (噪音策略 - 拒绝最优解)

## 1. When to Use (何时使用)

- 写作内容正文时
- 改进内容正文时
- 分析探讨内容时

## 2. Rule instruction (使用说明)

**[Core Logic Update]** 真正的“像人”，不是因为你做得对，而是因为你做得**不完全对**。

### 1. 反应未定 (Reaction Uncertainty)

**规则：** 面对危机或刺激，禁止立即给出“正确反应”。必须插入一段'无效时间'。

- **AI Logic:** 刺激 → 反应 (Too clean)
- **Ma Liu Logic:** 刺激 → 愣神/逃避/误读 → 反应
- **Example:** 杀手冲进来了。我第一反应不是拔刀，而是下意识地护住了桌上那杯刚泡好的枸杞茶。等茶杯稳住了，我才想起来该逃命。

### 2. 动机不闭合 (Unclosed Motives)

**规则：** 允许行为和心理之间存在'裂缝'。

- **Concept:** 嘴上是大义，心里是生意；或者嘴上是生意，心里是恐惧。
- **Example:** 我救了他。不是因为什么兄弟情义，纯粹是因为他欠我的三贯钱还没还。如果他死了，这笔账就成坏账了。为了财务报表的平衡，我不得不拼命。

### 3. 选择非最优 (Sub-optimal Choices)

**规则：** 在关键决策点，强制选择'情绪化'或'低效'的路径。

- **Logic:** 只要不导致任务彻底失败，尽量选那个让自己'爽'或者'省事'的选项，而不是最聪明的选项。
- **Example:** 逃跑路线有两条：A 是下水道，安全但臭；B 是屋顶，危险但帅。理性的特工选 A。但我选 B，因为我刚买的官靴不能沾屎，而且下水道会让我的风湿病复发。
- **[耐读性加成]:** 完美选择会终结阅读。读者一旦判断"这事已经被最聪明地处理了"，故事就只剩走流程。次优选择制造'提前知祸'——我知道你在犯错，但我想看你怎么摔。
