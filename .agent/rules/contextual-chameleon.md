# Rule: Contextual Chameleon (语境拟态 - 职场惊悚版)

## 1. When to Use (何时使用)

- 写作内容正文时
- 改进内容正文时
- 分析探讨内容时

## 2. Rule instruction (使用说明)

**[Universal Rule]** 无论什么背景，内核都是'社畜在赶工期'。

### 语境调优矩阵 (Context Calibration)

#### 1) ⚔️ 武侠 / 古代 / 宫廷
* **核心转化：** 衙门 = 大厂；圣旨 = 需求文档；大侠 = 外包团队。
* **名词库（精确+脏）：** 
    *   *精确：* 敕造、三省六部、不良人、坊墙、飞钱、驿站排单。
    *   *脏：* 霉斑、墨迹晕开、印泥发臭、公文纸受潮、官靴磨脚。
* **叙事逻辑：** “这事儿不合规矩，但为了赶在日落前结项，只能走个后门。”

#### 2) 🜁 玄幻 / 修仙
* **核心转化：** 宗门 = 垄断国企；渡劫 = 绩效考核；灵气 = 能源配额。
* **名词库（精确+脏）：** 
    *   *精确：* 筑基丹（批次号）、护山大阵（V3.0版）、灵石汇率、宗门贡献点。
    *   *脏：* 丹炉废渣、符纸烧焦味、灵兽粪便、飞剑生锈、打坐坐出的痔疮。
* **叙事逻辑：** “飞升是不可能飞升的，这辈子都不可能。光是凑齐渡劫用的避雷针材料，就得给炼器房那帮孙子送半年的礼。”

#### 3) 🚀 科幻 / 赛博朋克
* **核心转化：** 公司 = 极权政府；黑客 = 运维工程师；义体 = 廉价耗材。
* **名词库（精确+脏）：** 
    *   *精确：* 神经接口协议、散热阈值、多巴胺抑制剂、信用点汇率。
    *   *脏：* 硅脂干裂、接口积灰、机箱漏油、廉价塑料味、风扇噪音。
* **叙事逻辑：** “拯救世界？别逗了。我只是想在我的脑机接口彻底烧坏之前，把这单快递送完，换个二手的肾。”
