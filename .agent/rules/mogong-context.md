## ⚠️ 写作前强制检查清单

**在写作任何章节前，必须完成以下检查。未完成检查直接动笔将导致上下文错误。**

### 1. 确认当前章节位置

- [ ] 确认章节编号（如 C27）
- [ ] 确认章节所属卷（卷1/卷2/卷3）
- [ ] 阅读**前一章**正文结尾（至少最后 200 字）
- [ ] 阅读**当前章**大纲概要（包括 Hook、结账点）

### 2. 确认地理位置

| 卷 | 时间线 | 地理位置 | 说明 |
|----|--------|----------|------|
| 卷1 | C01-C10 | 宋鲁边境流民路 → 墨家边境营地 | 马六作为流民被收编 |
| 卷2 | C11-C26 | 宋鲁边境墨家营地（固定） | 驻扎半年，训练+日常 |
| 卷3 | C27+ | 边境 → 商丘（宋国都城） | 楚国攻宋，行军+守城 |

**⚠️ 严禁出现以下错误：**
- 卷2/3 中提到"返回鲁国"（他们一直在边境营地）
- 卷3 中距离商丘"八百里"（从边境到商丘约三百里）
- 卷2 中出现"芈夫人"（她在卷3中期才出场）

### 3. 确认人物状态

| 人物 | 卷1状态 | 卷2状态 | 卷3状态 |
|------|---------|---------|---------|
| 马六 | 流民/伪装男性 | 杂役/技术员/身份隐藏 | 墨子贴身/技术核心 |
| 墨子 | 初遇/收编 | 师徒/暧昧萌芽 | 关系升温/战场搭档 |
| 禽滑厘 | 监视者 | 勉强认可 | 保护者 |
| 曹公子 | 资源敌人 | 被打脸/合作 | 后勤配合 |
| 阿铁 | 迷弟 | 助手 | 战场帮手 |
| 芈夫人 | ❌不存在 | ❌不存在 | 中期登场 |
| 鲁班 | ❌不存在 | ❌不存在 | 后期对决 |

### 4. 确认术语规范

| 禁止 | 正确 | 说明 |
|------|------|------|
| 后勤组 | 辎重队 | 古代术语 |
| KPI/ROI（对话中） | 描述性表达 | 叙述层可用，对话层禁止 |
| 巨子（非弟子称呼） | 墨先生/墨翟 | 外人不称"巨子" |

### 5. 确认叙事视角

- [ ] **第三人称**：全文使用"马六"、"她"，禁止"我"
- [ ] **叙述层可用现代词**：KPI、成本核算、物理定律等
- [ ] **对话层禁用现代词**：角色只能说古人能说的话

## 结账协议检查

每章结尾必须包含：

1. **具体代价**：谁因她的选择付出了代价？
2. **不可逆性**：这个代价能修正吗？（必须是"不能"）
3. **她没说出口的话**：一句无法反驳、无法修正、无法再提起的事实

**示例：**
> 她没说出口的话：那碗粥是别人老婆的命。

## 常见错误速查

| 错误类型 | 示例 | 正确做法 |
|----------|------|----------|
| 地理错误 | "返回鲁国本营" | 他们一直在边境，从未离开 |
| 时间线穿越 | 卷2提到芈夫人 | 她在卷3中期才出场 |
| 距离错误 | "商丘还有八百里" | 从边境约三百里 |
| 人称错误 | "我觉得……" | "马六觉得……" |
| 对话污染 | 角色说"KPI" | 叙述层可用，对话层禁止 |
| 态度错误 | 弟子怀疑墨子 | 弟子对墨子绝对尊重 |
