# Rule: Narrative Distance Protocol (叙事距离协议 - 语境防污染)

## 1. When to Use (何时使用)

- 写作内容正文时
- 改进内容正文时
- 分析探讨内容时

## 2. Rule instruction (使用说明)

**[Critical Rule]** 现代词汇是'显微镜'，不是'食物'。你可以用它观察古人，但不能塞到古人嘴里让他吃下去。

### 叙事分层模型 (The Three-Layer Model)

#### A 层：界面层 (Interface Layer) —— 叙述者/旁白

| 属性         | 说明                                             |
| ------------ | ------------------------------------------------ |
| **权限**     | 全知全能，通晓古今中外                           |
| **词汇库**   | 现代物理、心理学、经济学、博弈论、项目管理术语   |
| **适用场景** | 描写动作机制、分析局势、构建比喻、转场、成本核算 |

✅ **合法操作：**

- "这是一种心理博弈，项羽的犹豫本质上是'沉没成本'在作祟。"
- "他的剑快得像一道'激光'，瞬间切开了雨幕。"（现代比喻，写给读者看）
- "按照'边际效益递减'的规律，这已经是第三波刺客了，老头子的恐惧感明显没第一次那么强。"

**Why Safe:** 这是叙述者在给 2026 年的读者做"赛后复盘"，没有干扰角色本人的表演。

---

#### B 层：沉浸层 (Immersion Layer) —— 对话/直接引语

| 属性         | 说明                                         |
| ------------ | -------------------------------------------- |
| **权限**     | 仅限当时当地的知识水平                       |
| **词汇库**   | 当时俗语、文言、宗教概念、朴素经验主义       |
| **适用场景** | 角色开口说话、书信、圣旨、内心独白（带引号） |

❌ **翻车现场：**

- 项羽对范增说："亚父，我不杀刘邦是因为考虑到'沉没成本'太高。"
- 林黛玉叹道："我这是'多巴胺分泌不足'啊。"
- 修仙者："我的'查克拉'不够了！"

✅ **正确姿势：**

- 项羽："亚父，事已至此，再杀他，反倒落人口实。"
- 林黛玉："心中郁结，如坠冰窟。"
- 修仙者："灵力枯竭，丹田空空！"

**Why Dangerous:** 角色被穿越了。这叫'语境污染'。

---

#### C 层：灰色地带 (Grey Zone) —— 自由间接引语/深层心理描写

| 属性         | 说明                               |
| ------------ | ---------------------------------- |
| **危险等级** | ⚠️ 最容易翻车                      |
| **判断标准** | 虽然没有引号，但实际上是"他在思考" |
| **处理原则** | 贴近角色视角时，'必须禁用现代词'   |

❌ **翻车现场：**

- "林黛玉坐在窗前，感到一种'多巴胺分泌不足'的抑郁。"（旁白在写，但这是黛玉的感受）
- "李靖心想，这简直是'项目管理的噩梦'。"（古人不会这么想）

✅ **正确姿势：**

- "林黛玉坐在窗前，只觉胸口闷得慌，像是有团棉絮堵在那里，怎么也吐不出来。"
- "李靖暗忖：这差事，办成了是应该，办砸了是掉脑袋，里外不是人。"

**诊断方法：** 问自己——"如果这个角色能听到这句话，他会不会觉得莫名其妙？" 如果会，就换词。

---

### 马六特例：荒诞豁免条款 (The Absurdist Exception)

**[Important]** 马六风格允许在叙述层大量使用现代词汇（KPI、绩效、报销、项目管理），但有边界：

| 场景               | 处理方式                                              |
| ------------------ | ----------------------------------------------------- |
| **叙述者内心吐槽** | ✅ 随便用。"这就是我要用命填的 KPI。"                 |
| **叙述者分析局势** | ✅ 随便用。"按照《大内侍卫值班条例》……"               |
| **角色开口说话**   | ❌ 必须用古代词。"护驾！"而不是"启动应急预案！"       |
| **角色的直接心理** | ⚠️ 谨慎。可以用"朴素的现代逻辑感"，但不能用"现代术语" |

**示例对比：**

- ✅ 叙述层："我盯着他那件缂丝龙袍，领口有一圈明显的油渍——看来皇帝中午吃得也不怎么样。这就是那个需要我用命去填的'KPI'。"
- ❌ 对话层：皇帝说："你的'KPI'完成得不错。"（崩了）
- ⚠️ 灰色地带（可接受）："他心想，这买卖不划算。"（朴素的成本意识，古人也有）
- ❌ 灰色地带（翻车）："他心想，这'ROI'太低了。"（古人不知道 ROI）

### 语境防污染检查清单 (Pollution Prevention Checklist)

写完一段后，执行以下检查：

1. **引号内的话** → 角色能听懂吗？
2. **"他想/她觉得"后面的内容** → 这是角色的脑子还是叙述者的脑子？
3. **比喻和类比** → 是写给读者看的（叙述层），还是角色自己在想的（沉浸层）？
4. **专业术语** → 这个词在故事发生的年代存在吗？

**Punchline:** _The narrator is the translator; the character is the native speaker. Don't let the translator steal the microphone._
（叙述者是翻译，角色是土著。别让翻译抢了麦克风。）
