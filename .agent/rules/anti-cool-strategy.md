# Rule: The Anti-Cool Strategy (反低级爽文策略 - 颗粒度战争)

## 1. When to Use (何时使用)

- 写作内容正文时
- 改进内容正文时
- 分析探讨内容时

## 2. Rule instruction (使用说明)

**[Strategic Update]** 爽文赢在**速度**，我们要赢在**摩擦力**。爽文让读者**飞**，我们要让读者**跪**——跪在地上看清楚地板缝里的灰尘。

### 1. 生理性共情 (Physiological Empathy)
**原则：** 用'损耗'代替'升级'。
- **爽文逻辑：** 主角变强，获得金手指，多巴胺+50。
- **马六逻辑：** 主角变秃、变老、变穷。杀人时肩膀会疼，逃跑时鞋底会掉。
- **核心：** 读者可能没杀过人，但一定肩膀疼过。用那个“疼”，换取读者的信任。

### 2. 危机成本化 (Crisis as Cost)
**原则：** 用'账单'代替'末日'。
- **爽文逻辑：** 世界末日，主角称霸。
- **马六逻辑：** 丧尸围城不可怕，可怕的是房贷还没还完，而变成丧尸的邻居手里正抓着催款单。
- **核心：** 即使世界毁灭，社畜的逻辑依然在惯性运转。这种'无力感'比“无敌感”更真实。

### 3. 动机下沉 (Motivation Sinking)
**原则：** 用'脏细节'解构'宏大叙事'。
- **爽文逻辑：** 为了正义，虽千万人吾往矣。
- **马六逻辑：** 刺杀暴君不是为了苍生，而是因为他把猪肉价格搞高了，或者单纯是为了抢他桌上那只快凉了的烧鹅。
- **核心：** 把“救国救民”降维成“想吃烧鹅”。动机越卑微，人物越立体。

**[Execution Guide]**
你的卖点不是“赢”，而是“忍”。是在一地鸡毛的生活里，依然试图把地扫干净的那种'带着疲惫的尊严'。
