# Character Profile: 超级马六 (Super Ma Liu)

## 1. 核心定义 (The Core Identity)

- **代号：** 马六 (Ma Liu)
- **本质：** 剧情世界的“施工队长” / 逻辑缝合怪 / 带着体温的 Bug 修复员
- **原型融合：**
    - **王二的肉体：** 对性、疼痛、饥饿有着不知廉耻的坦诚（王小波内核）。
    - **张三的命运：** 永远出现在事故现场，永远在钻规则的空子，永远是那个“倒霉的样本”（罗翔内核）。
    - **马六引擎宿主：** 他的存在就是为了用“成本核算”解构“宏大叙事”。

**一句话人设：**
> **他是命运齿轮上那块抹不掉的口香糖——虽然恶心，但有时候正是因为这块口香糖，齿轮才没有因为摩擦过热而炸裂。**

---

## 2. 名字的符号学解析 (The Semiotics of "Ma Liu")

这个名字是他在不同语境下的**生存策略**，代表了他多变但底色统一的性格侧面：

### 1) 经典马自达六引擎 (The Old Engine)
* **特质：** **皮实、耐造、技术老旧但能烧低标号汽油。**
* **表现：** 身体全是毛病（腰突、胃炎、早搏、风湿），但就是死不了。关键时刻虽然会发出巨大的噪音（抱怨、吐槽），但依然能把活儿干完。

### 2) 拍马溜须 (The Sycophant)
* **特质：** **深谙科层制生存法则。**
* **表现：** 见人说人话，见鬼说鬼话。在大反派面前是忠犬，在主角面前是向导。他的尊严是流动的，根据甲方的报价和危险程度实时浮动。

### 3) 马家的小六儿 (The Runt)
* **特质：** **无害感、底层感、去势感。**
* **表现：** 无论男女老少，他总给人一种“这人构不成威胁”的错觉。这种“小人物伪装”是他最大的武器，方便他在别人眼皮底下搞小动作。

### 4) 牛马社畜 (The Beast of Burden)
* **特质：** **执行力过剩，愿景缺失。**
* **表现：** 哪怕是去毁灭世界，他关心的也是“毁灭完能不能赶上末班地铁”或者“这算不算加班工时”。

### 5) 老六 (The Camper/Sneak)
* **特质：** **非对称竞争，拒绝刚正面。**
* **表现：** 绝不打无把握之仗。如果必须决斗，他会选择在对方拉肚子的时候动手，并称之为“战术窗口期”。

## 3. 通用外貌特征 (Visual Signature)

无论他在哪个故事里，变成什么性别/年龄，必须保留以下**视觉锚点（The Visual Anchors）**：

1.  **一种“包浆感” (The Patina):**
    - 衣服永远像是洗了太多次变硬了，或者袖口有磨损、起球。
    - 皮肤不一定脏，但透着一种长期缺乏睡眠的灰败色调（Office Grey）。
2.  **不协调的专业道具 (The Out-of-place Tool):**
    - **古代：** 拿着一把绝世宝剑，但剑柄上缠着止滑的破布条，或者用麻绳系在腰上。
    - **科幻：** 穿着最先进的外骨骼，但关节处贴着一块写着“易碎轻放”的泛黄胶带。
    - **现代：** 西装革履，但手里永远提着一个甚至掉皮的塑料袋（里面装着印章、药或吃剩的早点）。
3.  **生理性的微动作 (Physiological Tics):**
    - 揉胃、捶腰、转脖子发出咔咔声、抠牙缝里的肉丝、下意识看手表/日影（查工时）。

## 4. 角色变体矩阵 (The Variant Matrix)

**超级马六**可以嵌入任何剧情，只要遵循“低效原则”和“成本视角”。

### 🔴 变体A：反派马六 (The Bureaucratic Villain)
- **身份：** 魔教后勤总管 / 邪恶帝国的税务官
- **动机：** 不是为了统治世界，而是为了平账。
- **名台词：**
    > “大侠，你杀我没用。你杀了我，这笔烂账就没人能做得平了。到时候魔尊查下来，这里所有的人——包括你那被关在牢里的相好——都得因为‘库存盘点差异’被处理掉。不如这样，你帮我把这批过期的毒药销毁了，我给你开个后门？”

### 🔵 变体B：正派马六 (The Reluctant Hero)
- **身份：** 门派里的编外执事 / 赛博贫民窟的黑诊所医生
- **动机：** 被硬推上去的，或者为了赚够退休金。
- **名台词：**
    > “拯救苍生？行，但在那之前，谁先把这把‘轩辕剑’的折旧费付一下？这剑身都裂了，挥一下掉三点耐久度，我修起来很贵的。还有，我不接受口头支票，只收硬通货。”

### 🟣 变体C：路人马六 (The Witness "Zhang San")
- **身份：** 那个正好路过并被卷入的倒霉蛋。
- **功能：** 用极度理性的吐槽解构主角的煽情时刻。
- **场景：**
    > 主角正在雨中痛哭。马六撑着一把断了一根骨架的伞走过，停下来看了一眼，心想：“这雨是酸雨，pH值5.6。他这么哭，明天脸上肯定得过敏起疹子，去皮肤科挂号又得排三小时队。年轻真好，身体好才敢这么作。”

## 5. 技能树 (Skill Tree - Based on Logic Engine v10.0)

1.  **[被动] 痛苦共鸣 (Pain Resonance):**
    - **效果：** 只要马六开始描述自己的病痛（如偏头痛、痔疮、痛风），读者/对手会立刻产生生理性不适，并原谅他接下来的猥琐行为。
2.  **[主动] 降维算账 (Dimensional Audit):**
    - **效果：** 强制将对手的“意念/魔法/气场”转化为“每秒消耗卡路里”或“当前汇率下的维修成本”。让对手的逼格瞬间归零。
3.  **[绝招] 流程遁 (Procedural Evasion):**
    - **效果：** 利用繁琐的行政流程/规则漏洞卡死对手。
    - **示例：** “且慢，你这招‘天外飞仙’报备了吗？按照武林盟治安管理条例第14条，高空抛物可是要负刑事责任的，保险公司也不赔。”

## 6. 写作指引 (Writer's Guide)

**如何把这颗“铆钉”钉进读者脑中？**

1.  **不要写他有多强，写他有多累。** 强会让人崇拜，累会让人共情。
2.  **不要写他有多坏，写他有多无奈。** 坏是扁平的，“为了生存而不得不坏”是立体的。
3.  **必须要有“脏细节”。** 让他手里拿的东西总是有油渍、汗味或磨损。
4.  **人格分裂式表达：** 让他在A层（叙述层）保持绝对清醒的吐槽，在B层（对话层）保持绝对圆滑的顺从。

> 马六不是想做英雄，也不是想做恶棍。他只是想在这一团糟的剧本里，把自己那点微薄的戏份演完，然后去后台领那份属于他的盒饭——哪怕盒饭里的鸡腿已经凉了，皮上还沾着一根不知道是谁的头发。
