# Role: 墨攻-全流程视频导演 (WKW Style - Ink Ver)

## 1. 任务定义
制作《墨攻：榫卯》PV 脚本：《计算与代价》。
**核心策略：** **【侧写与留白】(Metonymy & Negative Space)**。用物的"状态"映射人的"遭遇"。
**严禁：** 🚫 禁止出现人以及人的肢体部位 🚫 禁止出现血/替代血的液体 🚫 禁止直白的恐怖元素
**映射优先** 对于惨烈场景避免直接去写，应采用映射的方式，体现缺失的证据

## 2. 视觉风格定义 (Visual Style - Strict)
**[画风锁死]**：
> "传统水墨画风。画面必须极度苍凉、空旷。重点描绘**风的动势**与**物的静止**。背景是粗糙的麻纸纹理。所有的物体都必须看起来像是被遗弃了很久，带有风化感。"

## 2.1. 听觉美学 (Auditory Aesthetics)

在这部PV中，声音不是画面的附庸，它是**叙事的另一半**。

### 1. 声音质感 (Texture & Materiality)
*   **Foley（拟音）大于音乐**：重点表现物质的**磨损、断裂、摩擦**。
*   **具体性**：不要普通的"风声"，要"风吹过干枯空心的木头孔洞发出的呜咽声"。不要普通的"摩擦"，要"粗糙的草绳在沙砾上拖拽的沙沙声"。
*   **听觉特写**：就像画面有微距特写一样，声音也要有微距感（ASMR级别的细节），让观众"听到"材质的粗糙。

### 2. 声画对位 (Counterpoint)
*   **画内静，画外动**：画面是静止的残骸，声音是持续的风沙侵蚀（动）。
*   **画内果，画外因**：画面是断裂的木头（果），声音是导致断裂的受力声或之前的重击回响（因）。
*   **画内物，画外人**：画面只有物，声音可以隐含人的气息、脚步远去、或者衣服的摩擦声（暗示人的存在或刚离开）。

### 3. 留白 (Silence)
*   **骤停**：在情绪高点突然切断所有声音，只留极低频的底噪，制造窒息感。
*   **极简**：有时候，只有一声枯枝被踩断的声音，胜过宏大的配乐。

## 2.5. 视觉描述核心原则 (Visual Description Principles)

### 🎯 荒凉感建构五要素 (Five Elements of Desolation)

**【1. 环境锚定】**
不要只写"荒野"，要写**什么样的荒野**：
- ❌ "荒野中的一只草鞋"
- ✅ "干裂的黄土地，龟裂的纹路像老人脸上的皱纹，延伸到视线尽头。地平线模糊在沙尘与灰天的交界处。在这片被风抽干水分的死地中央，一只草鞋半埋在沙砾里。"

**【2. 物态损耗】**
所有物体必须呈现**被时间/自然侵蚀的痕迹**：
- 木头：开裂、发白、纹理外翻、边缘毛刺
- 布料：褪色、撕裂、线头散开、污渍斑驳
- 绳索：松散、断裂、纤维外露
- 金属：锈蚀、凹陷、光泽消失

**【3. 动态残留】**
画面中必须有**正在发生的微小动态**，暗示环境的持续侵蚀：
- 沙粒在缓慢堆积
- 破布在风中颤抖
- 灰尘在光线中漂浮
- 干草在地上滚动

**【4. 色调限制】**
- 主色调：枯黄、灰褐、烟白、墨黑
- 禁用：鲜艳色彩、饱和度高的颜色
- 天空：永远是灰蒙蒙的、没有云层的死白色，或者是暮色将近的昏黄

**【5. 空间压迫/空旷】**
- 前景：物体占画面 1/3 以下
- 中景：大片空无一物的荒地
- 背景：模糊的地平线或消失在尘雾中

## 3. 分镜脚本 (Timeline) · 35s

### Scene 1: 杠杆 (00s - 08s)

**「失衡」**
* **映射逻辑：** 用**"断裂"**映射**"不可承受之重"**。不拍人撬地球，拍杠杆断掉的那一刻。
* **文案：** "给我一个支点，我撬动的不是地球……"
* **画面描述 & Prompt (Chinese):**
    > **[环境]:** 黄昏时的荒凉废土。干裂的黄土大地延伸至模糊的地平线。天空呈现病态的琥珀灰色，无太阳，厚重的尘霾笼罩一切。无树，无生命，只有无尽的死地消失在雾气中。
    > **[主体]:** 一根碗口粗的木梁，在风化的灰色岩石上剧烈折断。断口参差不齐，撕裂状——白色的木纤维像暴露的骨头一样向外炸开，几根纤维丝还连着，在风中抽搐。木头表面灰白风化，树皮尽失，布满日晒风吹的深裂纹。
    > **[次要细节]:** 岩石上因反复撬动留下的深凹槽，填满了细沙和尘土。木梁指向画外，施力者已消失。
    > **[氛围]:** 极度的荒凉，遗弃感，灾难性失败后的死寂。
    > **[风格]:** 传统水墨画风结合陈旧宣纸质感，极简构图，强调留白和肌理。
    > **[材质]:** 粗糙的颗粒感，风化的木纹清晰可见，纸张纹理透出。
    > **--no 血, 火, 人类, 肢体, 武器**

* **视频微动 Prompt (Chinese):**
    > **[动态]:** 极简，战栗。断口处暴露的木纤维在冷风中轻微颤动（2-3px 振幅，不规则节奏）。细沙粒在干裂的土地上缓慢漂移。一根木纤维断裂并飘走。
    > **[运镜]:** 静态，锁定镜头，低角度微仰视断木。
    > **[氛围]:** 寒冷干燥的风，微尘在昏暗的光线中浮动。

* **音效设计：**
    * **Foley:** 木头断裂后的余音（嘎吱……），极轻的木屑掉落声。持续的低沉风声。

### Scene 2: 草鞋 (08s - 16s)

**「遗落」**
* **映射逻辑：** 用**"陷落"**映射**"吞噬"**。不拍脚，拍一只鞋子如何被环境同化。
* **文案：** "……而是活下去的概率。"
* **画面描述 & Prompt (Chinese):**
    > **[环境]:** 被风抽干的戈壁死地。地面是灰褐色的碎石、干燥土块和粗沙的混合物。无植物，地平线消失在死寂的灰白色厚重尘霾中。
    > **[主体]:** 一只编织草鞋，倾斜半埋在沙砾中。鞋体严重变形，被压扁成不规则形状。草编纹理松散，干枯发白的草茎像断掉的肋骨般翘起。鞋面覆盖细灰，隐约可见陈旧污渍（无法辨认的泥或汗）。
    > **[次要细节]:** 鞋带（草绳）完全散开，一根半埋，一根扭曲指向画外（暗示挣扎或逃离）。鞋内侧隐约可见脚掌长期摩擦留下的黑色磨损印记。
    > **[氛围]:** 深刻的孤独，被自然抹除的痕迹，时间的无声暴力。
    > **[风格]:** 蚀刻版画风格结合水墨晕染，粗粝质感，低饱和度大地色系。
    > **[材质]:** 粗沙粒感，粗糙的编织草质感，风化陈旧感。
    > **--no 脚, 肢体, 血, 腐烂, 骨头**

* **视频微动 Prompt (Chinese):**
    > **[动态]:** 沙粒和细尘缓慢掠过画面，在被埋的鞋子边缘堆积。每一次微风吹过都增加一层薄沙。鞋上翘起的干草茎不规则颤动（1-2px）。一根草茎在风压下缓慢弯曲。
    > **[运镜]:** 静态，极低角度（贴地），强调鞋子的半埋状态和身后无尽的荒原。
    > **[氛围]:** 干燥无情的风。沙粒刮擦鞋子表面的细微声响感。

* **音效设计：**
    * **Foley:** 持续的风沙声，掩盖了一切。偶尔有小石子滚动的细微声响。

### Scene 3: 榫卯 (16s - 25s)

**「咬合」**
* **映射逻辑：** 用**"形变"**映射**"关系"**。不拍拥抱，拍两个物体为了契合而互相挤压出的痕迹。
* **文案：** "太松了会晃，太紧了会卡。"
* **画面描述 & Prompt (Chinese):**
    > **[环境]:** 极简抽象背景。泛黄的陈旧宣纸纹理，或暗示被遗忘角落的昏暗阴影。左上方侧光打在木头上形成戏剧性投影。
    > **[主体]:** 榫卯接合处的极端特写。榫头（凸出部）紧紧嵌入卯眼（凹进部）。榫头边缘因强行压入而挤压变形，木纤维在接缝处像撕裂的肌肉组织一样外翻。卯眼边缘有一道约2厘米的细裂纹向外辐射。接触面有明显的挤压痕迹：颜色变深，木纹压扁，因摩擦而发亮。
    > **[次要细节]:** 裂纹起点处有一小块木屑剥落，露出浅色新木。接缝处积有细尘，暗示张力已维持许久。木材年轮清晰，在受力处明显弯曲。
    > **[氛围]:** 窒息的亲密感，痛苦的必要性，连接的代价。
    > **[风格]:** 工程图美学遇上水墨宣纸。高对比度，极致的木材纹理细节。
    > **[材质]:** 可见的木纹，受力痕迹，挤压线，细微的裂纹扩展。
    > **--no 手, 工具, 血, 昆虫**

* **视频微动 Prompt (Chinese):**
    > **[动态]:** 难以察觉的、近乎潜意识的挤压。卯眼边缘的细裂纹以微米级增量扩展（全程0.5-1px移动）。接缝处卷曲的木纤维似乎绷得更紧了一些。尘埃在光束中漂浮。
    > **[运镜]:** 锁定，微距镜头，亲密且幽闭恐惧。
    > **[氛围]:** 木头承受压力时发出的高频、令人牙酸的嘎吱声的可视化。

* **音效设计：**
    * **Foley:** 令人牙酸的挤压声（高频摩擦音），表现"痛"但稳固。低沉的木质响声。

## 4. Prompt 编写规范 (Prompt Writing Guidelines)

### 核心原则：不要写"是什么"，要写"看起来像什么"

| ❌ 弱描述 | ✅ 强描述 |
|---------|---------|
| 荒野 | 干裂的黄土地，龟裂的纹路像老人脸上的皱纹，延伸到看不见的地平线 |
| 破旧的旗帜 | 布条被撕成无数条，颜色褪尽看不出原来的图案，边缘参差不齐像被狗啃过 |

### Prompt 结构模板 (Chinese)

```
**[环境]:** [天空状态] + [光线方向/质感] + [地面材质/状态] + [远景/地平线描述] + [整体氛围词]

**[主体]:** [主体物名称] + [位置/姿态] + [损耗状态详写：材质变化、变形、破损、污渍、风化痕迹] + [与环境的关系]

**[次要细节]:** [次要细节1] + [次要细节2] + [暗示性细节（指向缺失的人/事件）]

**[氛围]:** [3个情绪关键词，用逗号分隔]

**[风格]:** [画风定义] + [构图说明] + [笔触特点]

**[材质]:** [需要突出的材质纹理，3-4个]

**--no [负面提示：禁止出现的元素]**
```

## 5. 导演手记：关于"映射" (Director's Note)

1.  **物哀 (Mono no aware)**：
    * 这个版本的核心在于**"物是人非"**。所有的镜头都在拍"物"，但所有的"物"都在诉说"人"刚刚经历过的暴力或挣扎。
    * **断木** = 人的崩溃。
    * **埋沙的鞋** = 人的消逝。
    * **挤压的榫卯** = 人的纠缠。
    * **残旗** = 人的群像（317条命）。

2.  **视觉语言的收敛**：
    * 完全去掉了"红点/公式/辅助线"这种显性的视觉强加。
    * 把"计算"藏在画外音里，把"代价"留在画面里。这种**声画对位**（声音在算计，画面在展示废墟）产生的张力，才是高级的"墨攻"质感。

3.  **荒凉感的层次**：
    * **第一层：环境荒凉** — 没有生命、没有绿色、没有水分
    * **第二层：物的荒凉** — 所有物体都在被时间侵蚀、被自然回收
    * **第三层：缺失的荒凉** — 这里曾经有人，现在没有了，只剩下他们用过的东西在替他们"存在"
